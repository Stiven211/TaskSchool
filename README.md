# taskschool

Simplemente, app web 'TaskSchool' - lista de tareas y gestión.

Instrucciones rápidas:

1. Instalar dependencias: `npm install` (o `pnpm install` / `yarn`)
2. Ejecutar en dev: `npm run dev`
3. Construir: `npm run build` (salida en `dist/`)

Para subir a GitHub o desplegar en Vercel, proporciona la URL remota o un PAT con permisos `repo` y yo me encargo del push y de la configuración en Vercel.
